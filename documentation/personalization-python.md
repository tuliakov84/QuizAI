# Персонализация вопросов
### *- Выполнено на языке Python, расположено в ../src/main/python*

### Аргументы командной строки
<...>/src/main/python/service/question_personalization.py
- --candidates-file PATH - путь к JSON-файлу, хранящему кандидатов
- --target-count INT - количество вопросов после фильтрации
- --history-threshold FLOAT - максимальный коэффициент схожести с историей игроков
- --quiz-threshold FLOAT - максимальный коэффициент схожести внутри квиза
