llm.properties по умолчанию:

llm.model = gemma
llm.temperature = 0.2
numPredict = 2048
timeoutSeconds = 120

llm.model - выбор модели (по умолчанию gemma4)
llm.temperature - температура модели (по умолчанию gemma4)
numPredict - максимальное число токенов (по умолчанию 2048)
timeoutSeconds - таймаут в секундах (по умолчанию 120)

Любое из полей может отсутствовать, в таком случае будет установлено значение по умолчанию
